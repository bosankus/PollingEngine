PollHandle

interface PollHandle<T>

A live handle to a poll started with PollBuilder.start or PollBuilder.collect.

It carries the running poll's id and lets you pause, resume, cancel, or retune it directly — no need to look the session up by id after starting. Collect outcomes to await the terminal result of a converging poll.

val poll = Polling.poll { api.status() }.until { it == "DONE" }.every(2.seconds).start(scope)
poll.pause(); poll.resume(); poll.cancel()
